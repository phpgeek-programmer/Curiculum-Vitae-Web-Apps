about Me

Saya sangat tertarik dengan dunia komputer sejak sekolah menengah pertama, dengan mengikuti kursus word star, word perfect, lotus, dan dbase 3 plus. Dan mulai mendalami pemrograman komnputer secara otodidak dimulai dengan html, php dan javascript sebelum akhirnya mengambil kuliah di Universitas Islam Negeri Jakarta.

Tujuan Karir

Mengembangkan program aplikasi komputer berbasis web dan desktop dengan mengimplementasikan kode program yang simpel dan bersih. Menggunakan design pattern yang tepat untuk menghasilkan program yang handal, aman, dan mudah dikembangkan.

Keahlian

HTML5, CSS3 dan Javascript - Web Frontend dan UI

Mampu membuat halaman frontend aplikasi berbasis web dengan teknologi terkini yaitu: html5, css3, javascript dan framework JQuery. Bisa menggunakan framework CSS: 960 gs dan Twitter Bootstrap.

PHP - MySQL - Aplikasi web dengan database MySQL

Mampu membuat program aplikasi komputer berbasis web dengan bahasa pemrograman PHP dan menggunakan database MySQL. Mampu menggunakan framework PHP: Codeigniter, Laravel, dan Symfony 2.

Pemrograman Java - J2SE dan J2EE

Mampu membuat aplikasi berbasis desktop dengan J2SE dan mengembangkan aplikasi berbasis web dengan J2EE.

Linux - System Operasi untuk Server dan Desktop

Mampu bekerja dengan efisien dalam lingkungan linux maupun windows server. Bisa dan lebih menyukai bekerja dengan linux desktop seperti Ubuntu, ataupun fedora core.
